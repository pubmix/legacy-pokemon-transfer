# legacy-pokemon-transfer
This project allows the user to input a .sav file from Generation 1 (Red, Yellow, Blue) and Generation 2 (Gold, Silver, Crystal) and export save information.

Current Support: 

Gen I:
- Party Pokémon: yes
- PC boxes: not yet
- Full Gen I species name table: not yet

Gen II:
- Party Pokémon: yes
- PC boxes: yes
- Full Gen II species name table: not yet


DISCLAIMER: 
1. Current build does not support non-English formats of Gen 1/2 games. This will be covered in future builds.

2. User will need a FlashGBX connector to extract .sav files, as exporting original save content from original GB/GBC games is not supported in MRUpdater
